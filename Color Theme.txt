🔹 THEME NAME: Vaultecho 

🎯 DESIGN GOAL:
A dark-mode, security-focused, modern tech UI with cyber alert highlights using red accents and glowing interactions.

🧩 UI COMPONENTS STYLING:
━━━━━━━━━━━━━━━━━━━━
✅ Navbar:
- Transparent at top, sticky dark on scroll
- Hover underline animation
- Dropdown menu opens on hover
- Font-weight: 800; Letter-spacing: 0.5px

✅ Hero Section:
- Full screen with background image and dark overlay
- Large title with shadow
- Call-to-action buttons (red and white)
- Button hover: scale + shadow

✅ Feature Cards:
- Dark cards with red hover glow
- Rounded, modern design
- Icons animate on hover

✅ Safety Meter:
- Styled Bootstrap progress bar
- Red fill + rounded ends

✅ Stats Section:
- Large animated numbers
- Hover glow effects
- Highlighted titles

✅ VIP Quiz Box:
- Centered, rounded, neon-style glow
- Option selection with red pulses
- Result box with gradient icon badge

🔧 DEVELOPMENT TIPS:
━━━━━━━━━━━━━━━━━━━━
- Always use `var(--primary-red)` or `var(--dark-gray)` for consistency
- Animate using `fadeIn`, `wordRise`, `scale`, or `translateY`
- Prefer `.feature-card`, `.alert-card`, `.btn-hero` classes for reusability
- Maintain spacing using Bootstrap grids or `padding: 80px 0;`
